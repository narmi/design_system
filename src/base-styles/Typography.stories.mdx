# Typography

## Setting a typography context

Adding the `nds-typography` class to an element enables Narmi typesetting defaults
on child elements:

<Canvas isExpanded>
  <>
    <div className="nds-typography">
      <h2>Heading inside typography context</h2>
      <p>Paragraph inside of typography context</p>
    </div>
    <h2>Heading outside of typography context</h2>
    <p>Paragraph outside of typography context</p>
  </>
</Canvas>

## Elements styled by typography context

The `nds-typography` context targets heading and paragraph elements:

<Canvas isExpanded>
  <div className="nds-typography">
    <h1>Heading level 1</h1>
    <h2>Heading level 2</h2>
    <h3>Heading level 3</h3>
    <h4>Heading level 4</h4>
    <h5>Heading level 5</h5>
    <h6>Heading level 6</h6>
    <p>Paragraph</p>
  </div>
</Canvas>

## Typesetting with helper classes

This design system also provides [Helper Classes](?path=/docs/style-helper-classes--using-helper-classes)
to tweak style rules of individual elements. **These classes will work regardless of context**.

Font styles can be modified with classes following these signatures:

- `fontFamily--<family>`
- `fontSize--<size>`
- `fontColor--<color>`
- `fontWeight--<weight>`
